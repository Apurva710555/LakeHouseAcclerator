
## Steps to Run the App 

### 1. Databricks Workspace Access 
The user must have valid access to the Databricks workspace where the application is deployed.

### 2. Application Permissions 
Required permissions must be granted to the user or user group to allow access to the application.

### 3. Access via Databricks One 
Once access is provided, the user can open and access the application from Databricks One.

### 4. Application Status 
Ensure that the application is in a running state before attempting to access it.
